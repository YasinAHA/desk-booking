# Desk Booking

V1: reservas internas de escritorios (Madrid).

## Run local

1. Copia `app/config.example.js` a `app/config.js` y rellena tus keys.
2. Sirve la carpeta `app/` con un server local:

- Python:
  - `cd app`
  - `python -m http.server 5500`
  - abrir `http://localhost:5500`
