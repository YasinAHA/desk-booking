# ADR-0002: Commit and Tag Messages in English

- Status: Accepted
- Date: 2026-02-16
- Scope: core

## Context
El historial mostraba mezcla de idiomas en commits/tags, reduciendo consistencia y dificultando lectura uniforme en PRs y releases.

## Decision
Estandarizar mensajes de commit y descripcion de tags en inglés.

## Consequences
- Historial más consistente y legible.
- Mejor alineacion con herramientas y flujos habituales de desarrollo.
